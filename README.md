# Demo gestor de tareas

Monorepo con:
- Frontend: Vue 3.5+ + Vuetify
- Backend: Django + Django REST Framework
- Base de datos: Postgres
- Orquestación: Docker Compose

## Estructura

- /frontend: Aplicación Vue
- /backend: API Django
- /docs: Documentación y tareas

## Descripción

Este proyecto permite gestionar tareas pendientes (crear, marcar como completadas, eliminar completadas) siguiendo buenas prácticas de calidad y despliegue.
